## インストール

プロジェクトを開始するには、以下の手順に従ってください：

1. リポジトリをクローンします：

```zsh
# https
git clone https://github.com/yourusername/react-todo-app.git

# ssh
git clone git@github.com:ChanYou110/react-todo.git
```

2. dockerコンテナのビルド：

```zsh
docker-compose build
```

3. dockerコンテナの立ち上げ：

```zsh
docker-compose up
```
